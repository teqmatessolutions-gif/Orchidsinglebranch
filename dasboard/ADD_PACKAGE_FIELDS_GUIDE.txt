WHAT EACH FIELD DOES:
----------------------

1. THEME
   - Dropdown: Romance, Wellness, Adventure, Family, Business, Other
   - Helps categorize packages
   - Optional field

2. DEFAULT_ADULTS
   - Number input (default: 2)
   - Suggested number of adults for this package
   - Used as default when booking

3. DEFAULT_CHILDREN
   - Number input (default: 0)
   - Suggested number of children for this package
   - Used as default when booking

4. MAX_STAY_DAYS
   - Number input
   - Maximum number of days allowed for this package
   - Optional (null = unlimited)
   - Example: Honeymoon package might be max 7 days

5. FOOD_INCLUDED
   - Multi-select checkboxes: Breakfast, Lunch, Dinner, Snacks
   - Stored as comma-separated string
   - Shows what meals are included in the package
   - Example: "Breakfast,Dinner"
